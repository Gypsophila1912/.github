# .github リポジトリ

---

## 📁 ディレクトリ構成

```
.github/
├── ISSUE_TEMPLATE/
│   ├── bug_report.md
│   ├── feature_request.md
│   └── task.md
├── PULLREQUEST_TEMPLATE/
│   └── pullrequest_template.md
└── README.md
```

## 🐛 Issue テンプレート

### `ISSUE_TEMPLATE/bug_report.md`

**用途：** バグ報告

- 発生内容
- 再現手順
- 期待する挙動 / 実際の挙動
- 環境情報

を整理して書ける形式になっています。

---

### `ISSUE_TEMPLATE/feature_request.md`

**用途：** 機能要望・新機能提案

- 何をしたいか
- なぜ必要か
- どんな解決策を考えているか

を明確にするためのテンプレートです。

---

### `ISSUE_TEMPLATE/task.md`

**用途：** 作業タスク管理

- 目的
- 作業内容（チェックリスト）
- 完了条件

を整理して記載できます。

---

## 🔀 Pull Request テンプレート

### `PULLREQUEST_TEMPLATE/pullrequest_template.md`

PR 作成時に自動で読み込まれるテンプレートです。

- 変更内容
- なぜこの変更が必要か
- 動作確認内容
- 影響範囲

などを記載する想定です。
